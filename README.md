# FICSIT Standard Industrial Color System (SICS)

This repository hosts the official **FICSIT S.I.C.S. Portal**, a high-performance single-page application designed to manage factory aesthetics and wayfinding protocols in Satisfactory.

## Features

- **Master Swatch Registry:** A technical breakdown of the 18-slot Customizer configuration.
- **Operations Manual:** Strategic guidelines for functional zoning, signage, and structural finishes.
- **Interactive Library:** Instant copy-to-clipboard functionality for all 21 verified industrial hex codes.
- **Analytical Insights:** Data visualization of departmental color balance.

## Usage

1. Open the live URL provided by GitHub Pages.
2. Use the **Registry** tab to input hex codes into your game's Customizer.
3. Use the **Manual** tab to plan your factory expansion and logistics wayfinding.

## Technical Details

Built using:

- [Tailwind CSS](https://tailwindcss.com/) (UI/UX)
- [Chart.js](https://www.chartjs.org/) (Data Visualization)
- Vanilla JavaScript (State Management & Clipboard Logic)
